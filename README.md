# AUTO_LEARNING
機械学習において面倒臭いハイパラチューニングを担うライブラリです。予測モデル(Ridge, SVRなど)と、ハイパラ候補(Ridgeであれば`alpha=[0.0001, 0.001 0.1, 1, 10, 100, 1000]`など)を指定すると、ハイパラチューニングを行い、予測結果を返します。ハイパラ探索には、全探索とベイズ最適化の2つの方法があり、指定できます。
